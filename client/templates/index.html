OpenFX, a format built for everyone

OpenFX is an open standard for image processing plug-ins. It allows plug-ins written to the standard to work on any application that supports the standard. This avoids the current per application fragmentation of plugin development and support, which causes much heartache to everyone: plug-in developers, application developers and end-users alike.

About ShuttleOFX

A project

The OFX platform allows to share OpenFX plugins between plugin creators and graphic artists. In the continuity of the TuttleOFX initiative, the ambition is to promote the emergence of a standard way to create image processing algorithms usable across softwares. The project was initially proposed by Fabien Castan and Marc-Antoine Arnaud to a team of three students from the IMAC engineering school in Paris, FRANCE.
