Welcome on board

The ShuttleOFX platform is designed to share OpenFX plugins between plugins creators (industrials, developpers, researchers, students, ...) and graphic artists.
Following the TuttleOFX initiative, the ambition is to promote a standard way to create image processing algorithms usable across softwares.

A format built for everyone

OpenFXOpen standard

OpenFX is an open standard for image processing plug-ins. It allows plug-ins written to the standard to work on any application that supports the standard. This avoids the current per application fragmentation of plugin development and support, which causes much heartache to everyone: plug-in developers, application developers and end-users alike.
